Under the Hood

Senior Engineer Decisions

Provider-Agnostic LLM

One env var switches between Gemma 4, Llama, or Gemini. Zero code changes.

Named AMD Step

generateEmbedding() is explicitly visible in the workflow. Not buried in generic code.

Graceful Fallbacks

AMD down? → Gemini. Gemma deployment down? → Llama serverless. Never crashes.

Convex-First Backend

All logic in Convex queries/mutations/actions. Reactive by default, zero WebSocket code.

LangChain StateGraph

Workflow orchestration inside a Convex action. Classify → Draft → Escalate as a DAG.

Auto-Send Architecture

Auto-replies send immediately. Escalated emails wait for human. One workflow, two paths.
